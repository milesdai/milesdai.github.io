# Personal Site

## Setup

```bash
# Clone with submodules...

git clone --recurse-submodules git@github.com:milesdai/milesdai.github.io.git

# ...or manually initialize submodules

git clone git@github.com:milesdai/milesdai.github.io.git
git submodule init
git submodule update

# Install Hugo and all deps

# Start Hugo server
hugo server
```
